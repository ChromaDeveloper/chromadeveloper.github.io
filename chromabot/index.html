What's ChromaBot ?

Basically...

  • ChromaBot is an open source Discord bot that is built using Python, Discord.py, and discord.ext. It is designed to help with moderation tasks on Discord servers, such as kicking, warning, banning, and purging messages. In addition, it has many other useful commands that you can discover by using the c!help command. We highly recommend giving ChromaBot a try on your Discord server.
  • Also do not forget that it's in the BETA stages. The stable release will be there soon.
